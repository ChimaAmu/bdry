This is a simple diary for those of us who spend probably too much time in the terminal

## Usage
* -a (or -add) a new entry line on the current date
* -d (or -delete) {line} to delete the specified line
* -p (or -print) to print entry of specified date (current if not specified)
* -e (or -edit) {old} {new} replace words in entries

## Features
* Dates are auto-sorted into Year/Month folders
